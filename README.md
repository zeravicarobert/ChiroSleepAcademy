# ChiroSleepAcademy

Working repository for **Chiropractic Sleep Academy (CSA)** — a category-building
project teaching chiropractors how to build a credible sleep-and-airway
division around **Structural Airway Chiropractic**: a skull-down, sleep-first
model that integrates patient education, structural correction (including
VAET), home sleep testing, CBCT imaging, and ethical collaboration with
dentists, ENTs, and sleep physicians.

**Owner:** Robert Zeravica, DC
**Domain:** [www.ChiropracticSleepAcademy.net](https://www.ChiropracticSleepAcademy.net)
**Site platform:** ClickFunnels 2.0

## What lives here

This repo is the long-term workspace for CSA planning and execution.

- `CLAUDE.md` — project memory: mission, audience, voice, offer, pricing, goals
- `MAY_2026_GOALS.md` — the active monthly tracker
- `outreach/` — outreach drafts, templates, and partner communications
- `site/` — public site content (placeholder; site itself is hosted on ClickFunnels 2.0)
- `content/` — course and educational assets
